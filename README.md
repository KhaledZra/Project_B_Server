# Project B Server
Game server for client!

CMake tutorial how to build and run


## build and run from scratch
- mkdir test_build
- cd .\test_build\
- cmake ../../Project_B_Server
- cmake --build .
- cd Debug
- .\Project_B_Server.exe


## build and run from existing build
- cmake --build .
- cd Debug
- .\Project_B_Server.exe
